# Big Data Sentiment Analysis

## Introduction

Big Data Sentiment Analysis is the process of analyzing text data to determine whether the sentiment is positive, negative, or neutral.

This project performs sentiment analysis on sample text data using Python and TextBlob library.

---

## Objectives

- To analyze text sentiments
- To classify sentiments as positive, negative, or neutral
- To use Python libraries for sentiment analysis
- To understand text processing techniques

---

## Tools Used

- Python
- Pandas
- TextBlob
- Google Colab
- GitHub

---

## Steps Performed

1. Installed TextBlob library
2. Imported required libraries
3. Created sample text dataset
4. Performed sentiment analysis
5. Classified sentiments into Positive, Negative, and Neutral

---

## Conclusion

The project successfully analyzed text sentiments using the TextBlob library. Positive, Negative, and Neutral sentiments were identified from text data, demonstrating how sentiment analysis helps understand user opinions and feedback.
